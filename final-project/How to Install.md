# How to run the project (Bash Script)

## Run Script

```bash
bash run.sh
```

## Open Browser

```bash
open http://localhost:4173
```

# How to run the project (Command by Command)

## Install Node.js

```bash
nvm install
```

## Use Node.js

```bash
nvm use
```

## Install Dependencies

```bash
npm install
```

## Build Project

```bash
npm run build
```

## Start Project

```bash
npm run preview
```

## Open Browser

```bash
open http://localhost:4173
```
